void *  context 
)

This function compares sizeToRead bytes between flashAddress and the buffer provided by expBuff.

Returns
zero on success.
A non-zero value on failure.
Parameters
[in]flashAddressThe flash address to compare to.
[in]expBuffThe expected buffer. Its size must be at least sizeToRead bytes.
[in]sizeToReadThe number of bytes to compare.
[in,out]contextUser context.
