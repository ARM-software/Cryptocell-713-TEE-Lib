Detailed Description

Contains functions that access the OTP memory for read and write operations.

Function Documentation

◆ CC_BsvOTPWordRead()

CCError_t CC_BsvOTPWordRead ( unsigned long  hwBaseAddress,
uint32_t  otpAddress,
uint32_t *  pOtpWord 
)

This function retrieves a 32-bit OTP memory word from a given address.

Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressThe base address of the CryptoCell HW registers.
[in]otpAddressThe address of the word in the OTP memory to read from.
[out]pOtpWordThe data read from the word in otpAddress in OTP memory.

◆ CC_BsvOTPWordWrite()

CCError_t CC_BsvOTPWordWrite ( unsigned long  hwBaseAddress,
uint32_t  otpAddress,
uint32_t  otpWord 
)

This function writes a 32-bit OTP memory word to a given address. After writing, the word is read and compared to the "otpWord".

Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressThe base address of the CryptoCell HW registers.
[in]otpAddressThe address of the word in the OTP memory to write to.
[in]otpWordThe data to write to the word in otpAddress in OTP memory.
