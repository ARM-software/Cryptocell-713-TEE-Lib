Function Documentation

◆ CC_PalVerifyCertValidity()

uint32_t CC_PalVerifyCertValidity ( char *  pNotBeforeStr,
uint32_t  notBeforeStrSize,
uint8_t  notBeforeStrFlag,
char *  pNotAfterStr,
uint32_t  notAfterStrSize,
uint8_t  notAfterStrFlag 
)

This function checks validity period and should be implemented by customer. It receives start and end validity period as input. It also receives an indication flag for each period. If the flag is not 1, the value of current period was not defined by the user.

Returns
CC_OK On success.
A non-zero value on failure.
Parameters
[in]pNotBeforeStrPointer to the start period string.
[in]notBeforeStrSizeSize of the start period string.
[in]notBeforeStrFlagStart period definition flag indication.
[in]pNotAfterStrPointer to the end period string.
[in]notAfterStrSizeSize of the end period string.
[in]notAfterStrFlagStart period definition Flag indication.
