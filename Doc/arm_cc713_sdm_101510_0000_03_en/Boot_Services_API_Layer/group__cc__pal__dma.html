Detailed Description

Contains the functions that are used for the SBROM PAL DMA layer.

Function Documentation

uint32_t CC_PalDmaMap ( CCAddr_t  origDataAddr,
CCAddr_t *  mappedDataAddr,
uint32_t  dataSize,
CCPalDmaBufferDirection_t  dmaDirection,
CC_PalDmaBufferHandle  dmaHandle 
)

This API can be used to flush buffers to physical memory, handle cache-line alignment issue (by copying the buffers), or virtual to physical addresses translation, prior to passing a DMA-able buffer address to an SBROM API for processing.

Returns
Returns 0 if successful and other value if not
Parameters
[in]origDataAddrAddress of the original buffer (can be virtual).
[out]mappedDataAddrMapped data address (must be physical).
[in]dataSizeSize of the buffer in bytes.
[in]dmaDirectionThe required direction of the copy operation, if a copy to a temporary buffer is required.
[in,out]dmaHandleOptional handle if additional data is needed for the unmap operation. Can be NULL.

void
