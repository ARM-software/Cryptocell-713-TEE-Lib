CCError_t CC_BsvRsaPrimVerify ( unsigned long  hwBaseAddress,

This function performs the primitive operation of RSA, meaning exponent and modulus. outBuff = (pInBuff ^ Exp) mod NBuff. ( Exp = 0x10001 )

The function supports 2k and 3K bit size of modulus, based on compile time define. There are no restriction on pInBuff location, however its size must be equal to BSV_RSA_KEY_SIZE_IN_BYTES and its value must be smaller than the modulus.

Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressThe base address of the CryptoCell HW registers.
[in]NBuffThe modulus buffer big endian format.
[in]NpBuffThe barret tag buffer big endian format - optional.
[in]pInBuffThe data-in buffer to be encrypted.
[in]inBuffSizeThe data-in buffer size in bytes, must be BSV_RSA_KEY_SIZE_IN_BYTES.
[out]pOutBuffThe encrypted buffer in big endian format.
[in]pWorkSpaceThe pointer to user allocated buffer for internal use.
[in]workBufferSizeThe size in bytes of pWorkSpace, must be at-least BSV_RSA_WORKSPACE_MIN_SIZE.

This function performs RSA PSS verify. 

The function should support 2k and 3K bit size of modulus, based on compile time define.
Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressThe base address of the CryptoCell HW registers.
[in]NBuffThe modulus buffer big endian format.
[in]NpBuffThe barret tag buffer big endian format - optional.
[in]signatureThe signature buffer to verify - big endian format.
[in]hashedDataThe data-in buffer to be verified as sha256 digest.
[in]pWorkSpaceThe pointer to user allocated buffer for internal use.
[in]workBufferSizeThe size in bytes of pWorkSpace, must be at-least BSV_RSA_WORKSPACE_MIN_SIZE.
[out]pIsVerifiedThe flag indicates whether the signature is verified or not. If verified value will be CC_TRUE, otherwise CC_FALSE
