◆ CC_BsvProductionPkgVerify()

CCError_t CC_BsvProductionPkgVerify ( unsigned long  hwBaseAddress,
unsigned long  pkgLoadAddr,
unsigned long *  pImageCodeAddr 
)

This function decrypts and authenticates the production image package. The decryption is done in-place. After successful verification the function outputs the image address so the boot flow can jump too.

Note
pkgLoadAddr must be equal to the value defined in the scatter file while building the CMPU/DMPU images
Returns
CC_OK on success.
A non-zero value from bsv_error.h on failure.
Parameters
[in]hwBaseAddressCryptoCell HW registers' base address.
[in]pkgLoadAddrProduction image load address. 0 address is not allowed. address of allocated buffer should be large enough to contain the entire encrypted package
[out]pImageCodeAddrThe image code address that the boot flow should jump to.
