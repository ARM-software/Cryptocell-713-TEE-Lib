Detailed Description

A structure to define key exchange context. All byte arrays in this structure are stored in the big-endian byte ordering, and all word arrays are in the little endian byte and word ordering.

◆ confirmation

uint8_t CC_Sm2KeContext_t::confirmation

The first bit encodes whether this party wants a confirmation. The second bit encodes the confirmation for other party for example: 3 - for both partys, 1 - only this party wants confirmation, 2 - only the other party wants confirmation.
