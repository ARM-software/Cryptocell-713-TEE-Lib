◆ CC_Des()

CIMPORT_C CCError_t CC_Des ( CCDesIv_t  IV_ptr,
CCDesKey_t *  Key_ptr,
CCDesNumOfKeys_t  NumOfKeys,
CCDesEncryptMode_t  EncryptDecryptFlag,
CCDesOperationMode_t  OperationMode,
uint8_t *  DataIn_ptr,
size_t  DataInSize,
uint8_t *  DataOut_ptr 
)

This function is used to operate the DES machine in one integrated operation.

Returns
CC_OK on success.
A non-zero value from cc_des_error.h on failure.
Parameters
[in]IV_ptrThe IV buffer in CBC mode. In ECB mode this parameter is not used.
[in]Key_ptrPointer to the user's key buffer.
[in]NumOfKeysThe number of keys used: single (56bit), double (112bit) or triple (168bit).
[in]EncryptDecryptFlagA flag that determines if the DES should perform an Encrypt operation (0) or a Decrypt operation (1).
[in]OperationModeThe operation mode: ECB or CBC.
[in]DataIn_ptrThe pointer to the input data. The size of the scatter/gather list representing the data buffer is limited to 128 entries, and the size of each entry is limited to 64KB (fragments larger than 64KB are broken into fragments <= 64KB).
[in]DataInSizeThe size of the input data. Must be a multiple of the DES block size, 8 bytes.
[out]DataOut_ptrThe pointer to the output data. The size of the scatter/gather list representing the data buffer is limited to 128 entries, and the size of each entry is limited to 64KB (fragments larger than 64KB are broken into fragments <= 64KB).

◆ CC_DesBlock()

CIMPORT_C CCError_t CC_DesBlock ( CCDesUserContext_t *  ContextID_ptr,
uint8_t *  DataIn_ptr,
size_t  DataInSize,
uint8_t *  DataOut_ptr 
)

This function is used to process a block on the DES machine. This function should be called after the CC_DesInit function was called.

Returns
CC_OK on success.
A non-zero value from cc_des_error.h on failure.
Parameters
[in]ContextID_ptrPointer to the DES context buffer allocated by the user, which is used for the DES machine operation. This should be the same context used on the previous call of this session.
[in]DataIn_ptrThe pointer to input data. The size of the scatter/gather list representing the data buffer is limited to 128 entries, and the size of each entry is limited to 64KB (fragments larger than 64KB are broken into fragments <= 64KB).
[in]DataInSizeThe size of the input data. Must be a multiple of the DES block size, 8 bytes.
[out]DataOut_ptrThe pointer to the output data. The size of the scatter/gather list representing the data buffer is limited to 128 entries, and the size of each entry is limited to 64KB (fragments larger than 64KB are broken into fragments <= 64KB).

◆ CC_DesFree()

CIMPORT_C CCError_t CC_DesFree ( CCDesUserContext_t *  ContextID_ptr)

This function is used to end the DES processing session. It is the last function called for the DES process.

Returns
CC_OK on success.
A non-zero value from cc_des_error.h on failure.
Parameters
[in]ContextID_ptrPointer to the DES context buffer allocated by the user that is used for the DES machine operation. This should be the same context that was used on the previous call of this session.

◆ CC_DesInit()

CIMPORT_C CCError_t CC_DesInit ( CCDesUserContext_t *  ContextID_ptr,
CCDesIv_t  IV_ptr,
CCDesKey_t *  Key_ptr,
CCDesNumOfKeys_t  NumOfKeys,
CCDesEncryptMode_t  EncryptDecryptFlag,
CCDesOperationMode_t  OperationMode 
)

This function is used to initialize the DES machine. To operate the DES machine, this should be the first function called.

Returns
CC_OK on success.
A non-zero value from cc_des_error.h on failure.
Parameters
[in]ContextID_ptrPointer to the DES context buffer allocated by the user, which is used for the DES machine operation.
[in]IV_ptrThe IV buffer. In ECB mode this parameter is not used. In CBC this parameter should contain the IV values.
[in]Key_ptrPointer to the key buffer of the user.
[in]NumOfKeysThe number of keys used: 1, 2 or 3 (defined by the enum). One key implies DES encryption/decryption, two or three keys imply triple-DES.
[in]EncryptDecryptFlagA flag that determines whether the DES should perform an Encrypt operation (0) or a Decrypt operation (1).
[in]OperationModeThe operation mode: ECB or CBC.
