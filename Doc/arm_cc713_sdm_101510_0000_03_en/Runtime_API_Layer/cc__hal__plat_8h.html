Write CryptoCell memory-mapped-IO register.

Note
This macro must be modified to make the operation synchronous:
  • The write operation must complete.
  • The new value must be written to the register before the macro returns.
The mechanisms required to achieve this are architecture-dependent (for example: the memory barrier in Arm architecture).
Parameters
regOffsetThe offset of the Arm CryptoCell register to write.
valThe value to write.
