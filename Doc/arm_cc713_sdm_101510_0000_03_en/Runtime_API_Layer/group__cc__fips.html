◆ CC_FipsIrqHandle()

CCError_t CC_FipsIrqHandle ( void  )

This function is used to handle the interrupt that is issued when the CryptoCell-713 REE is updating its FIPS status.

Note
This function should be integrated into the Interrupt Handler Routine of the operating system of the user. In the example implementation, this function is called through a CC_InterruptHandler.
Returns
CC_OK on success,
A non-zero value from cc_fips_error.h on failure.
