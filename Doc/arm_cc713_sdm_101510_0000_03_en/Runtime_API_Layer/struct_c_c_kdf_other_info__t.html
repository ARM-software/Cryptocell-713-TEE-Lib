Detailed Description

KDF structure, containing pointers to OtherInfo data entries and sizes.

The structure contains two arrays: one for data pointers and one for sizes. They are placed according to the order given in the ANSI X9.42-2003: Public Key Cryptography for the Financial Services Industry: Agreement of Symmetric Keys Using Discrete Logarithm Cryptography standard and defined in CCKdfOtherInfoEntries_t enumerator. In KDF ASN1 mode this order is mandatory. In other KDF modes, you can insert optional OtherInfo simply in one (preferably the first) or in some entries. If any data entry is not used, then the pointer value and the size must be set to NULL.
