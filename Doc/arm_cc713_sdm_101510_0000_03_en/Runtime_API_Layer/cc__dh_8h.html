Detailed Description

This file defines the API that supports Diffie-Hellman key exchange, as defined in Public-Key Cryptography Standards (PKCS) #3: Diffie-Hellman Key Agreement Standard and in ANSI X9.42-2003: Public Key Cryptography for the Financial Services Industry: Agreement of Symmetric Keys Using Discrete Logarithm Cryptography (key lengths 1024 and 2048 bits).
