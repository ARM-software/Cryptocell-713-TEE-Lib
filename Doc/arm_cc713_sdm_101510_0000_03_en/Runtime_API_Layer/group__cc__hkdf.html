CC_HkdfKeyDerivFunc performs the HMAC-based key derivation, according to RFC5869.

Returns
CC_OK on success.
A non-zero value on failure as defined cc_kdf_error.h, cc_hash_error or cc_hmac_error.h
Parameters
[in]hkdf_hash_modeThe HKDF identifier of hash function to be used.
[in]salt_ptrA pointer to a non secret random value. can be NULL.
[in]saltLenThe size of the salt_ptr.
[in]ikm_ptrA pointer to a input key message.
[in]ikm_lenThe size of the input key message
[in]info_ptrA pointer to an optional context and application specific information. can be NULL
[in]info_lenThe size of the information.
[out]okmA pointer to a output key material.
[in]okm_lenThe size of the output key material.
[in]is_strong_keyif TRUE , then no need to perform the extraction phase.
