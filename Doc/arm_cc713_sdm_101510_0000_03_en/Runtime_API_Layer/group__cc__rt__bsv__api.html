◆ CC_SbrtCertChainVerificationInit()

CCError_t CC_SbrtCertChainVerificationInit ( CCSbCertInfo_t *  certPkgInfo)

This function initializes the Secure Boot certificate chain process.

This function initializes the structure that holds the state of the certificate chain.

Note
This must be the first API called when processing the Secure Boot certificate chain.
Returns
CC_OK on success.
A non-zero value from bootimagesverifier_error.h on failure.
Parameters
[in,out]certPkgInfoA pointer to the structure that holds the state of the certificate chain.

◆ CC_SbrtCertVerifySingle()

CCError_t CC_SbrtCertVerifySingle ( CCSbFlashReadFunc  flashReadFunc,
void *  userContext,
CCAddr_t  certSrcAddress,
CCSbCertInfo_t *  certPkgInfo,
CCSbX509TBSHeader_t *  pX509Header,
uint32_t *  pWorkspace,
uint32_t  workspaceSize,
CCSbImagesInfo_t *  pImagesInfo,
CCSbUserAddData_t *  pUserData 
)

This function verifies a single key or content certificate.

This function verifies the following details:

  • The public key that is saved in the certificate, against its hash that is either found in the OTP memory (Hbk) or in certPkgInfo.
  • The RSA signature of the certificate.
  • The NV counter in the certificate is higher than or equal to the minimum NV counter, as recorded on the device and passed in certPkgInfo.
  • Each software module against its hash in the certificate, for content certificates.
Returns
CC_OK on success.
A non-zero value from bootimagesverifier_error.h on failure.
Parameters
[in]flashReadFuncA pointer to the flash read function.
[in]userContextAn additional pointer for flash read usage. It may be NULL.
[in]certSrcAddressThe flash address where the certificate is located. This address is provided to flashReadFunc.
[in,out]certPkgInfoA pointer to the structure that holds the state of the certificate chain.
[in,out]pX509HeaderA pointer to X509 TBS header information.
  • For proprietary format, this parameter is not applicable (n/a).
  • For X509 format: this parameter is optional. If pX509Header->pBuffer points to legal buffer (not NULL), the X509 TBS header information will be copied to it.
[in]pWorkspaceA buffer for internal use by the function.
[in]workspaceSizeThe size of the workspace, in bytes. Must be at least CC_SB_MIN_WORKSPACE_SIZE_IN_BYTES.
[out]pImagesInfoA pointer to the information about the verified images, which includes addresses and sizes.
[out]pUserDataA buffer holding the user additional data. The buffer is being used only when CC_CONFIG_BSV_CERT_WITH_USER_ADDITIONAL_DATA compilation flag is set.

◆ CC_SbrtGetCertSize()

CCError_t CC_SbrtGetCertSize ( CCSbCertChainType_t  chainType,
uint32_t *  pCert,
uint32_t *  pCertSizeWords 
)

This function returns the size in words of a certificate.

Note
The certificate (or part of it which includes the header) should be loaded to the RAM prior calling this function.
This API is not relevant for X509 run time Secure Boot solution.
Returns
CC_OK on success.
A non-zero value from bootimagesverifier_error.h on failure.
Parameters
[in]chainTypeThe certificate chain type: CC_SECURE_BOOT_CHAIN / CC_SECURE_DEBUG_CHAIN
[in]pCertThe certificate address (after it has been loaded to memory).
[in,out]pCertSizeWordsA pointer to certificate size in words:
  • in: The size of the data which has been loaded to memory.
  • out: The actual certificate size.

◆ CC_SbrtSwImageStoreAddrChange()

CCError_t CC_SbrtSwImageStoreAddrChange ( uint32_t *  pCert,
uint32_t  maxCertSizeWords,
CCAddr_t  address,
uint32_t  indexOfAddress 
)

This function changes the storage address of a specific software image in the content certificate.

Note
The certificate must be loaded to the RAM and verified prior to calling this function.
This API is not relevant for X509 SB solution.
Returns
CC_OK on success.
A non-zero value from bootimagesverifier_error.h on failure.
Parameters
[in]pCertThe certificate address (after it has been loaded to memory).
[in]maxCertSizeWordsThe certificate boundaries - the maximal memory size allocated for the certificate, in words.
[in]addressThe new storage address to change to.
[in]indexOfAddressThe index of the software image in the content certificate, starting from 0.
