The function builds (imports) the ECC Domain structure from EC parameters given by the user in big-endian order of bytes in arrays.

Call this function first when you operate the ECC cryptographic operations. The function performs the following operations:

  • Checks pointers and sizes of incoming parameters.
  • Converts parameters from big-endian bytes arrays into little endian words arrays, where the left word is the last significant.
Note
Domain parameters should be validated by the user, prior to calling this function.
Returns
CC_OK on success.
A non-zero value on failure as defined cc_ecpki_error.h.
Parameters
[in]pModA pointer to EC modulus.
[in]pAA pointer to parameter A of elliptic curve. The size of the buffer must be the same as EC modulus.
[in]pBA pointer to parameter B of elliptic curve. The size of the buffer must be the same as EC modulus.
[in]pOrdA pointer to order of generator (point G).
[in]pGxA pointer to coordinate X of generator G. The size of the buffer must be the same as EC modulus.
[in]pGyA pointer to coordinate Y of generator G. The size of the buffer must be the same as EC modulus.
[in]pCofA pointer to EC cofactor - optional. If the pointer and the size are set to null, the given curve has cofactor = 1 or cofactor should not be included in the calculations.
[in]modSizeBytesA size of the EC modulus buffer in bytes.
Note
The sizes of the buffers: pA, pB, pGx, pGx are equal to pMod size.
Parameters
[in]ordSizeBytesA size of the generator order in bytes.
[in]cofSizeBytesA size of cofactor buffer in bytes. According to our implementation cofactorSizeBytes must be not great, than 4 bytes. If cofactor = 1, then, the size and the pointer may be set to null.
[in]securityStrengthBitsOptional security strength level S in bits: see section A.3.1.4 of ANSI X9.62-2005: Public Key Cryptography for the Financial Services Industry, The Elliptic Curve Digital Signature Algorithm (ECDSA). If this parameter is equal to 0, then it is ignored, else the function checks the EC order size. If the order is less than max(S-1, 192), then the function returns an error.
[out]pDomainA pointer to EC domain structure.
