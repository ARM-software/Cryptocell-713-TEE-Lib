◆ TEST_CALC_BUFF_ZEROS

#define TEST_CALC_BUFF_ZEROS (   wordBuf,
  buffWordSize,
  zeros 
)
Value:
{\
uint32_t i = 0;\
uint32_t j = 0;\
uint32_t mask = 0;\
zeros = 0;\
for (i = 0; i< buffWordSize; i++) {\
for (j = 0; j< BITS_IN_32BIT_WORD; j++) {\
mask = 0x1;\
if (!(*(wordBuf+i) & (mask << j))) {\
zeros++;\
}\
}\
}\
}

Test calculation buffer zeros.
